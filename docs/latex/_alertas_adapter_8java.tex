\doxysubsection{Descripción detallada}
Adaptador para gestionar y mostrar una lista de alertas en un Recycler\+View. 

Este adaptador enlaza los datos de las alertas con las vistas de cada elemento en el Recycler\+View, permitiendo también manejar eventos como la eliminación de alertas.
