Este proyecto es una aplicación Android que utiliza beacons para medir niveles de ozono y temperatura. La aplicación se conecta a un servidor para almacenar y recuperar datos de sensores a través de una API REST.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Funcionalidades}{Funcionalidades}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Detección de Dispositivos BTLE}}\+: La aplicación busca y se conecta a beacons Bluetooth Low Energy.
\item {\bfseries{Medición de Sensores}}\+: Recopila datos de sensores (ozono y temperatura) y los envía al servidor.
\item {\bfseries{Gestión de Usuarios}}\+: Permite la creación y eliminación de usuarios en el sistema.
\item {\bfseries{Interfaz de Usuario}}\+: Interfaz sencilla para interactuar con la aplicación y visualizar datos.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Requisitos}{Requisitos}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Android Studio
\item SDK de Android (versión mínima recomendada\+: 21)
\item Dependencias\+:
\begin{DoxyItemize}
\item Retrofit para la comunicación con la API REST
\item Gson para la conversión de JSON
\end{DoxyItemize}
