\doxysubsection{Descripción detallada}
Clase que representa la trama de un beacon i\+Beacon. 

Esta clase se encarga de interpretar los datos de un beacon i\+Beacon a partir de un array de bytes. Extrae información relevante como el UUID, el major, el minor y el Tx\+Power.
