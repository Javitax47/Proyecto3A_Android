\doxysubsection{Descripción detallada}
Clase que implementa la actividad para mostrar y gestionar alertas. 

La actividad utiliza un Recycler\+View para mostrar alertas que se reciben mediante un Broadcast\+Receiver. También incluye funcionalidades para borrar alertas y gestionar el menú.
