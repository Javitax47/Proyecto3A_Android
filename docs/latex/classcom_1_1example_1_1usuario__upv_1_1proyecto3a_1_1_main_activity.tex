Actualiza la dirección IP del servidor para la conexión con la API. 

Este método es llamado cuando el usuario introduce una IP y presiona el botón para actualizarla. Valida que la IP no esté vacía, construye la URL base para las peticiones HTTP mediante Retrofit y actualiza la instancia de la API. Muestra un mensaje en pantalla indicando la nueva IP del servidor.

Procesa la información del beacon detectado. 

Este método determina el tipo de medición a partir del valor \textquotesingle{}major\textquotesingle{} del beacon y registra el valor \textquotesingle{}minor\textquotesingle{}. Los tipos de medición reconocidos son CO2 y Temperatura, identificados por sus respectivos códigos.


\begin{DoxyParams}{Parámetros}
{\em major} & El valor \textquotesingle{}major\textquotesingle{} del beacon, que contiene el tipo de medición y un contador. \\
\hline
{\em minor} & El valor \textquotesingle{}minor\textquotesingle{} del beacon, que representa el dato medido (por ejemplo, CO2 o temperatura). \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Devuelve}
Un entero que representa el tipo de medición\+:
\begin{DoxyItemize}
\item 1 si el dato es de CO2
\item 2 si el dato es de temperatura
\item 0 si el tipo de dato no es reconocido. 
\end{DoxyItemize}
\end{DoxyReturn}
