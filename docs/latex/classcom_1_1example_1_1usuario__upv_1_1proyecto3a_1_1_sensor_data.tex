\doxysubsection{Descripción detallada}
Clase que representa los datos de un sensor. 

Esta clase se utiliza para almacenar la información de las mediciones de los sensores, incluyendo el sensor\+\_\+id, el valor, la marca de tiempo, el ID del usuario asociado, y el tipo de sensor.
